In definitions, always use the first rule that applies.
So, you probably know the 2-variable Veblen function, ϕ(α,β), or ϕα(β), but I won't write that, since it'll be used for something more powerful.

Definition:
φ(0,α) = ωα
φ(β,α) = (1+α)th point of γ↦φ(δ,γ) for all δ<β (β>0)

But you can also extend the function to 3 variables, like φ(α,β,γ). So like, φ(1,0,0) = Γ0 = φ(φ(φ(...,0),0),0), φ(1,0,1) =
Γ1 = φ(φ(φ(...φ(φ(1,0,0)+1,0)...,0),0),0), φ(1,1,0) = φ(1,0,φ(1,0,φ(1,0,...))), φ(2,0,0) = φ(1,φ(1,φ(1,...,0),0),0)

Definition:
φ(0,0,α) = ωα
φ(γ,β,α) = (1+α)th fixed point of δ↦φ(γ,η,δ) for all η<β (β>0)
φ(γ,0,α) = (1+α)th fixed point of δ↦φ(η,δ) for all η<γ (γ>0)

Of course, you can have φ(,...,α3,α2,α1,α0).

Definition:
Let Zk denote a list of k zeros. Let # denote an arbitrary list of ordinals φ(α) = ωα
φ(Zk,#) = φ(#)
φ(#,β,Zk,α) = (1+α)th point of γ↦φ(#,δ,γ,Zk-1,0) for all δ<β (β>0)
