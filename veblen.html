In definitions, always use the first rule that applies.
So, you probably know the 2-variable Veblen function, φ(α,β), or φα(β), but I won't write that, since it'll be used for something more powerful.

Definition:
φ(0,α) = ωα
φ(β,α) = (1+α)th point of γ↦φ(δ,γ) for all δ<β (β>0)

But you can also extend the function to 3 variables, like φ(α,β,γ). Like this: φ(1,0,0) = Γ0 = φ(φ(φ(...,0),0),0) = φ(0,φ(0,φ(0,...,0),0),0), φ(1,0,1) = Γ1 = φ(φ(φ(...φ(φ(1,0,0)+1,0)...,0),0),0), φ(1,1,0) = φ(1,0,φ(1,0,φ(1,0,...))), φ(2,0,0) = φ(1,φ(1,φ(1,...,0),0),0), φ(ω+1,0,0) = φ(ω,φ(ω,φ(ω,...,0),0),0), etc.

Definition:
φ(0,0,α) = ωα
φ(γ,β,α) = (1+α)th fixed point of δ↦φ(γ,η,δ) for all η<β (β>0)
φ(γ,0,α) = (1+α)th fixed point of δ↦φ(η,δ) for all η<γ (γ>0)

Of course, you can have φ(,...,α3,α2,α1,α0). So φ(1,0,0,0) = φ(φ(φ(...,0,0),0,0),0,0), φ(1,1,0,0) = φ(1,0,φ(1,0,φ(1,0,...,0),0),0), φ(2,0,0,0) = φ(1,φ(1,φ(1,...,0,0),0,0),0,0), φ(1,0,0,0,0) = φ(φ(φ(...,0,0,0),0,0,0),0,0,0), etc.

Definition:
Let Zk denote a list of k zeros.
Let # denote an arbitrary list of ordinals
k is an integer greater than -1.
φ(α) = ωα
φ(Zk,#) = φ(#)
φ(#,β,Zk,α) = (1+α)th point of γ↦φ(#,δ,γ,Zk-1,0) for all δ<β (β>0)

The limit of this, φ(1,...,0,0,0) is the Small Veblen ordinal, equal to ψ0(ΩΩω) in Buchholz's function. To go further, we need a notation for writing numbers at transfinite positions. Here is the common one: (α_nβ_n ... α_2β_2 α_1β_1 α_0β_0) means α0 at position β0 (counting from 0 and starting from the left), α1 at position β1, etc, until αn at position βn, where n is finite.

Definition:
... can be anything, even empty.
In a (α_nβ_n ... α_2β_2 α_1β_1 α_0β_0), any columns of the form {0α} are removed (any references to them in the rules return 0.) φ(α0) = ωα
φ(...αβ γ0) = (1+γ)th point of δ↦φ(...ηβ δλ ) for all η<α, λ<β

The limit of this, φ(1φ(1...)), is the Large Veblen Ordinal
