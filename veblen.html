So, you probably know the 2-variable Veblen function, ϕ(α,β), or ϕα(β), but I won't write that, since it'll be used for something more powerful.

Definition:
ϕ(0,α) = ωα
ϕ(β,α) = (1+α)th point of γ&maps;ϕ(δ,γ) for all δ<β (β>0)

But you can also extend the function to 3 variables, like ϕ(α,β,γ). It is defined like so:

Definition:
ϕ(0,0,α) = ωα
ϕ(γ,β,α) = Least fixed point of δ&maps;ϕ(γ,η,δ) for all η<β (β>0)
ϕ(β,0,α) = (1+α)th fixed point of δ&maps;ϕ(η,δ) for all η<γ (γ>0)
