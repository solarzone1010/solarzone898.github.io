So, you probably know the 2-variable Veblen function, ϕ(α,β), or ϕα(β), but I won't write that, since it'll be used for something more powerful.

Definition:
φ(0,α) = ωα
φ(β,α) = (1+α)th point of γ↦φ(δ,γ) for all δ<β (β>0)

But you can also extend the function to 3 variables, like φ(α,β,γ). It is defined like so:

Definition:
φ(0,0,α) = ωα
φ(γ,β,α) = Least fixed point of δ↦φ(γ,η,δ) for all η<β (β>0)
φ(β,0,α) = (1+α)th fixed point of δ↦φ(η,δ) for all η<γ (γ>0)
